ข้อที่ 1 : ให้เขียน code robot framework โดยใช้ library Builtin โดยให้ทำตาม​โจทย์ดังนี้
    ให้สร้างโจทย์บวกเลข โดยกำหนด veriable มา 2 ตัว ไว้ใส่ค่าตัวเลข หลังจากนั้นให้ทำการกำหนดค่าให้ variable ทั้งสองตัว และทำการ expect ค่าที่ออกมา 
    ตัวอย่างเช่น
        a + b = c
        a = 10
        b = 10
        a + b = 20
        และทำการ expect ว่า c ต้อง = 20
        โดยต้องมีการ log ค่าของ c มาให้ดูด้วย
